# Spectralization TODO

本文档是路线图，不是承诺清单。它记录当前能玩什么、哪些系统处在原型期、哪些功能只保留设计方向。

状态标记：

- `[x]` 已实现，并至少经过基础验证。
- `[~]` 原型可用，但仍需要数值、体验、性能或回归测试。
- `[ ]` 尚未实现。
- `[!]` 已知问题或需要重新设计。
- `[hold]` 保留想法，暂时不进入当前阶段。

## 当前阶段

当前阶段目标是把已经接入的基础系统收束成中期玩法。优先级从高到低：

1. 稳定光学网络编译器、日志、调试入口和 HUD 可视化。
2. 完成微缩机器第一版玩法闭环。
3. 让全息存储矩阵成为可用的中期存储系统。
4. 补齐生存模式早期光源、光学玻璃/掺杂材料路线和数据包文档。
5. 开始微缩化光路工作站、超材料、透镜材料和后续光刻路线。

当前不优先做：

- 真理套装终局能力。
- 光剑等武器线。
- 全世界复振幅干涉。
- 高复杂度像差、单模/多模光纤细节损耗、科研级光学仿真。
