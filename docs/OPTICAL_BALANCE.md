# Spectralization 数值平衡草案

本文档记录 beta0.0.1 前后的光学功率、泵浦、电力和材料参数目标。这里的数值不是现实单位，而是玩法单位；它们的任务是让时代边界、软锁和工程优化都有可解释的量级。

## 1. 单位

| 单位 | 含义 | 玩法意义 |
| --- | --- | --- |
| SP | Spectral Power，有效相干光功率 | 激光加工、光刻、微缩、虚空前置等机器的核心输入 |
| PU | Pump Unit，有效泵浦强度 | 增益介质把散杂光转成相干光，并放大相干光的内部资源 |
| FE | 普通电力 | 玩家熟悉的外部能量输入，转化为 PU 后参与激光工程 |

SP 不等于全部光功率。散杂光、热光、无效频率和烂光束可以有很大总功率，但不会直接满足精密机器的 SP 门槛。

## 2. 基准红宝石激光器

第一代标定机器：

| 条件 | 数值 |
| --- | ---: |
| 红宝石块数量 | 5 |
| 每个红宝石的有效泵浦源上限 | 4 |
| 总有效泵浦 | 20 PU |
| 背镜 | 银块或银涂层高反射面 |
| 输出镜 | 涂银玻璃 |
| 目标稳定输出 | 100 SP |

设计解释：

- 6 面全泵浦会让方块无法自然串联成增益柱，因此普通工程上限按 4 个有效泵浦源计算。
- 100 SP 是外部可用输出，不是红宝石内部产生的源功率。
- 在输出镜透射率约 0.24、反馈调度目标约 0.94 的情况下，腔增强量级约为 `0.24 / (1 - 0.94) = 4`。
- 因此内部红宝石相干源预算的目标量级是 `25 SP`，也就是约 `1.25 SP / PU`。

## 3. FE 到 PU

FE 到 PU 的效率缓慢提升，但 PU 与 SP 需求随时代快速上升。

| 时代 | 建议转化率 | 典型描述 |
| --- | ---: | --- |
| 激光器前 | 无稳定 PU | 只能做散杂光、检测、辐光发电和表面工程 |
| 早期固体激光器 | 200 FE/t = 1 PU | 萤石、岩浆块、红石和粗糙泵浦结构 |
| 改良固体激光器 | 150 FE/t = 1 PU | 更好涂层、光纤耦合、较稳定腔 |
| 光刻机时代 | 110 FE/t = 1 PU | 工程化泵浦和更好晶体 |
| 微缩时代 | 75 FE/t = 1 PU | 微腔、超材料和高效耦合 |
| 黑洞时代 | 45 FE/t = 1 PU | 高效率，但机器需求进入指数级 |

用早期红宝石硬堆到光刻机门槛是允许的，但电力、空间、热、光束质量和稳定性成本都会明显超过当前时代。

## 4. 阶段功率

| 阶段 | 典型有效 SP | 典型 FE/t | 时代目标 |
| --- | ---: | ---: | --- |
| 激光器前 | 0 coherent SP | 2-500 | 发电、光检测、涂层、散杂光源 |
| 第一台红宝石激光器 | 10-50 | 800-3000 | 首次产生相干光 |
| 标准红宝石柱 | 100 | 4000 | 5 红宝石、20 PU 的参考机器 |
| 优化红宝石阵列 | 300-1000 | 20000-100000 | 多腔、合束、改良涂层 |
| 一代光刻机 | 2000 | 200000+ | 优质激光和光束质量硬门槛 |
| 二代光刻机 | 10000 | 1000000+ | 更好晶体、镀膜和谱线控制 |
| 三代光刻机 | 50000 | 5000000+ | 微缩结构前置 |
| 微缩机器 | 200000-5000000 | 20000000-500000000 | 方块压缩、端口密度、稳定性软锁 |
| 黑洞时代 | 10000000+ | 1000000000+ | 不稳定微缩机器和高能激光保护 |

精密机器读取的是：

```text
effectiveSP = rawSP * spectralMatch * beamQuality * stability * coupling
```

所以高总功率不必然进入新时代。玩家可以暴力堆机器，但越接近硬门槛，光谱、光束质量和可靠性越重要。

## 5. 机器门槛

| 机器 | 最低输入 | 理想输入 | 锁类型 |
| --- | ---: | ---: | --- |
| 激光诱导相变台 | 20 SP | 100 SP | 软锁 |
| 早期切割/烧蚀 | 50 SP | 300 SP | 软锁 |
| 光纤耦合器 | 100 SP + 光束质量 | 500 SP | 半软锁 |
| 一代光刻机 | 2000 SP | 3000 SP | 硬时代门槛 |
| 二代光刻机 | 10000 SP | 15000 SP | 硬时代门槛 |
| 三代光刻机 | 50000 SP | 80000 SP | 微缩前置 |
| 微缩压缩机 | 200000 SP | 1000000 SP | 能量软锁 |
| 稳定黑洞前置 | 10000000 SP | 50000000 SP | 终局门槛 |

## 6. 发电与储能

| 系统 | 时代 | 单元输出目标 |
| --- | --- | ---: |
| 微型辐光收集器 + 火把 | 激光器前 | 2-4 FE/t |
| 微型辐光收集器 + 萤石/海晶灯 | 激光器前 | 8-16 FE/t |
| 堆叠式辐光阵列 | 激光器前/固体激光器 | 64-512 FE/t |
| 光腔发电结构 | 固体激光器 | 512-4096 FE/t |
| 光热塔/光腔发电机 | 光刻机前后 | 8000-64000 FE/t |
| 高级光能储存与发电阵列 | 微缩时代 | 100000 FE/t 以上 |

小型发电机允许堆叠。大型结构不以“不能堆”为卖点，而以腔体、几何、冷却和材料效率倍率为卖点。

## 7. 前中期材料定位

| 材料 | 阶段 | 用途 | 光学倾向 |
| --- | --- | --- | --- |
| SiO2 / 玻璃 | 激光器前 | 透镜、窗口、早期光路 | 可见高透射，低反射，低吸收 |
| Ag / 银 | 激光器前后 | 早期高反射镜 | 可见和近红外高反射，低吸收 |
| 涂银玻璃 | 固体激光器 | 输出镜 | 镀银轴部分透射、强反射，侧向接近玻璃 |
| Au / 金 | 固体激光器 | 红外反射、特殊镀层 | 红/红外好，蓝紫损耗更高 |
| Cu / 铜 | 固体激光器 | 热接口、低级反射 | 中等反射，高吸收，适合光热 |
| Fe / 铁 | 激光器前后 | 结构、早期挡光/反射 | 反射中等，吸收较高 |
| TiO2 / 金红石 | 固体激光器 | 高折射涂层前体 | 强散射/反射，高损耗，不适合直接通光 |
| Al2O3 / 刚玉 | 固体激光器 | 晶体基底 | 可见较高透射，紫外损耗较高 |
| Cr:Al2O3 / 红宝石 | 固体激光器 | 第一代增益介质 | 红宝石线低损耗，其他可见波段吸收较强 |
| CaF2 / 氟石 | 固体/光刻 | 紫外窗口、低色散透镜 | 从可见到紫外都高透射 |
| Y2O3 / 氧化钇 | 光刻前置 | YAG 来源 | 主要是材料链节点 |
| YAG | 光刻时代 | 高级晶体基底 | 高稳定、低损耗、可掺杂 |
| Nd:YAG | 光刻时代 | 强固体激光 | 比红宝石效率高 |
| Yb:YAG | 光刻/微缩 | 高效率泵浦激光 | 高功率路线 |
| Er:YAG | 中后期 | 红外加工 | 特殊材料加工 |
| Ti:Al2O3 | 光刻/后期 | 可调谐激光 | 宽谱、昂贵 |
| YAG:Ce | 中期 | 荧光与转换材料 | 散杂光转换 |
| Si / 硅 | 光刻时代 | 芯片、光纤工程 | 进入光刻工业链 |
| He-Ne | 中期 | 稳定参考激光 | 校准、测量、仪器 |

## 8. 第一轮代码参数目标

当前代码先落地已经会参与自由空间光路的材料：

| 材料 | 当前目标 |
| --- | --- |
| 玻璃 | 可见高透射，不再只有单点常数 |
| 水 | 修正可见 bin 上限，保留红外强吸收 |
| 银块 | 高反射，作为早期背镜 |
| 金/铜/铁块 | 分离不同金属 profile，不再共用一个粗略 metal |
| 红宝石块 | 红宝石线高透射，非目标可见波段高吸收；泵浦后提供相干源和相干增益 |
| 金红石块 | 高折射/强散射材料的粗略功率级表现 |
| 刚玉块 | 未掺杂晶体基底，较高透射 |
| 氟石块 | 紫外/可见高透射 |

后续材料会逐步迁移到数据包 LUT。代码常数只是 beta0.0.1 的硬编码原型。
